MarkNotes

MarkNotes

Use markdown to write notes for:

  1. easy to write, do not need to bother to the format;
  2. easy to reuse the resources including equations, codes, figures et al.;
  3. easy to read.

将创作初期的文档用Markdown格式编写，并汇总在MarkNotes文件夹下。如此可以解决Markdown插入的资源分散的问题，同时也方便后续的修改。

用于准备正是文档，而非备忘录。

Markdown 擅长记录的内容：公式、在线图片、代码片段及类似 Markdown 语言绘制的图形。这些正好可以弥补 OneNote 的不足。因此，我将 MarkNote 作为 OneNote 的一个补充。

Notebooks

Format for the name of the notes: [BOOK]_[SECTION]_[PAGE]

Setup subfolders for the notebooks.

Development

与项目对应的说明文档。

Papers

文章写作的大纲，描述方法的公式，写作进度与写作日志。

该笔记为 MarkNotes 中最重要的一个。

Reference

记录可能会引用到的文献，按照文献的 publisher 进行分类。

收藏文献中的公式、图片（在线）和摘录比较有用的句子。

Blog

用于书写发布到网上的博客内容。

Instructions

Insert figures

  • 每个Notebook的figures保存在子文件夹Figs中；
  • 插入其他Notebook中的figures可以采用以下几种方式：
    • 拷贝至当前笔记本的Figs文件夹中；
    • Onedrive同步后获取共享链接；
    • 采用原位置的绝对路径，这种方式会因为路径更改导致图片失效。
  • 插入在线图片的链接，图片会自动显示，显示出 Markdown 基于 Web 的特性。
