WSL display desktop

WSL 显示桌面

总结起来有两种方式在 Windows 上显示 Linux 桌面，Ubuntu和Kali都没问题。

方式一：Windows端运行 Xserver 程序如 VcXsrv，Linux 端设置 DISPLAY=:0.0并运行xfce4-session即可。

方式二：Linux端启动 /etc/init.d/xrdpstart，Windows 端通过远程桌面连接127.0.0.1:3390 即可显示桌面。
